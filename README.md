## Spelling Checker Application Using Python

## Introduction
This project is a simple desktop based spelling checker application developed using Python.
It uses a graphical user interface to allow users to enter text with spelling mistakes and get corrected output instantly using natural language processing.

## Features
Graphical user interface using Tkinter  
Real time spelling correction  
Simple and beginner friendly design  
Uses NLP based word correction  

## Technologies Used
Python  
Tkinter  
TextBlob  

## Project Structure
main.py  
Contains GUI code and spelling correction logic  

## How It Works
User enters incorrect spelling in the input box  
User clicks the Check button  
TextBlob processes the input text  
Corrected spelling is shown in the output box  

## Code Explanation
The application uses TextBlob to correct spelling.
Tkinter is used to build the desktop interface.
The correct_spelling function reads user input applies correction and updates the result field.
